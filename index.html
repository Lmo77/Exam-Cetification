How to set up your Google Sheet

1
Your sheet needs these column headers in row 1: word | type | cefr | topic | definition | example | translation_es | image — the image column is optional; leave it blank for words without one.
2
File → Share → Publish to web → choose your sheet tab → choose Comma-separated values (.csv) → click Publish → copy the URL.
3
Paste the URL below, press Test, then copy the app URL and share it with students. They choose their own level, topics, and activities.

How to set up your Medical Google Sheet

1
Main vocab tab — row 1 headers must be exactly: word | type | monthly_theme | subtopic | definition_en | example_en | spanish
2
Config tab (a second tab, for the weekly banner) — row 1 headers: key | value. Then two rows: active_theme → this month's theme, and active_subtopic → this week's subtopic. Update these cells each week to change the banner.
3
File → Share → Publish to web → publish each tab separately as .csv → paste both URLs below.
